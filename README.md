# MusicPlayer2

一款功能强大的 Windows 本地音乐播放器，提供歌词显示、频谱分析、专辑封面管理等丰富功能，支持高度自定义的界面布局。

## 功能特性

### 播放与音频
- 支持多种音频格式（MP3、FLAC、APE、WAV、WMA、OGG 等）
- 双播放内核：BASS / FFmpeg
- 均衡器与混响音效设置
- CUE 分轨支持
- 播放速度与变调调节
- 格式转换工具

### 歌词与标签
- 歌词卡拉OK样式显示
- 桌面歌词与迷你模式歌词
- Windows 10/11 搜索框歌词集成
- 歌词在线下载与编辑
- 歌曲标签（ID3）识别与编辑
- 专辑封面在线下载与管理

### 界面与体验
- 多种预设界面布局（UI1-UI5、Groove音乐风格等）
- 深色/浅色模式
- 主题颜色自定义
- XML 自定义界面布局
- 背景高斯模糊效果
- 任务栏缩略图按钮
- 媒体库管理
- 播放队列面板
